# TidyPackRat Assets

This directory contains visual assets for the TidyPackRat project.

## Logo
- `logo.png` - Main project logo (pack rat mascot with glasses)
- Used in README.md and documentation

## Icon
- `icon.ico` - Windows application icon (multiple sizes embedded)
- Used by the GUI application

## Screenshots
- `screenshots/` - Application screenshots for documentation
- Add screenshots of:
  - Main configuration window
  - File categories view
  - Schedule configuration
  - Test run in action

## Installer Graphics (Optional)
- `Banner.bmp` (493×58 pixels) - Installer banner
- `Dialog.bmp` (493×312 pixels) - Installer dialog background
